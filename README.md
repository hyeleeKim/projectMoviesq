# project_movie (2023.05.12 ~ 23.08.17)
아이디어 구상, 아이템 지정 -> 기획(요구사항 정의, UML, 와이어프레임, DB 설계, API 명세) -> 개발(프론트,백엔드)  -> 배포

- 김혜리 (팀원)
- 박도현 (팀원)
- 이상민 (팀원)

## 목표
